Projects

MyGrades

Single Page Application

Built with Javascript, AngularJS & Google Firebase

MyGrades is an application I built for students to keep track of, and make calculations on, the grades they receive in their college or high school courses. The "What Do I Need" feature allows a student to calculate a grade that they need to get on an upcoming assignment (homework, exam, project, etc.) in order to get a specific grade (chosen by the user) in the course based on previous grades in the course. MyGrades was built using the Angular framework, with a hint of jQuery. MyGrades stores data using Google's Firebase platform, and was styled with Bootstrap.

Djangazon

E-Commerce Web Application

Built with Python, the Django Framework & SQLite3

I worked on this project in a team of 4 people, and we followed the SCRUM Sprint methodology. We worked on this project for two week-long sprints. The Djangazon project is an e-commerce web application, similar to Amazon, that was built using the Django framework. Users can buy and sell products, like/dislike products, give ratings to purchased products, and recommend products to friends (via username). My team and I also built a recommendation engine that recommends products to users, based on the Jaccard Index's similarity coefficient (relative to other user's like/dislike patterns). Check it out on GitHub!

An immersive, full-time boot-camp that trains full-stack software developers for 6 months.


During the first three-month session I created and contributed to many different projects. I built and styled static web apps with vanilla Javascript, HTML and CSS.
